SharedPreferenceService

SharedPreferenceService class Null safety

Utility class for SharedPreferences.

It creates a singleton in order to have it in the whole widget tree without calling SharedPreferences.getInstance() multiple times. Easily modifiable to hold maybe a volatile cache of the preferences.
